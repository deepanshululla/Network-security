This is project submission for attacks on TCP/IP using either Netwox with C or scapy with python.

The list of attacks includes 
1) ARP cache poisoning
2) ICMP Redirect Attack
3) SYN Flooding Attack
4) TCP RST Attacks on telnet and SSH Connections
5) TCP RST Attacks on Video Streaming Applications
6) ICMP Blind Connection-Reset and Source-Quench Attacks
7) TCP Session Hijacking 

Analyzed packets in wire shark to determine why certain attacks were not successful in this OS.
